This is a version specifically packaged for Windows 7 systems. After differenceList.txt is generated, close the exe program and disable your proxy, then run the exe again to download resources normally.
Python stopped supporting Windows 7 from version 3.9.5 onwards, so to package for Windows 7, the lower version 3.8.10 must be used. However, the network library in version 3.8.10 has very poor proxy support!
To use a proxy would require various configurations, so for convenience, the only option is to use a workaround: first generate differenceList.txt, then disable the proxy and download resources directly.

If you encounter issues, try turning your proxy on or off. I have no other solutions.
