# Flower-knight-girl-Viewer_update

Update tool for Flower Knight Girl Viewer. Place it in the package.nw directory (same level as the scenes directory) to update.

If you encounter errors when running the update program, try turning your proxy on or off.

For the latest version, please download from the tags section. The programs in the win7 and win10 directories are older versions.

**The program prioritizes updating scenes based on the IDs in differenceList.txt. If you don't need this, please delete differenceList.txt before running the update program (if differenceList.txt is outdated, it will automatically crawl the wiki to generate a new file).**

## Folder Structure

- **win7** / **win10**: Executables for different Windows versions
- **全场景翻译(至2023-11-16)** (Full Scene Translation - up to 2023-11-16): Complete scene translations using GPT-3.5-turbo
- **全场景翻译(至2024-01-18)** (Full Scene Translation - up to 2024-01-18): Updated translations with improved quality
- **剧本转换程序** (Script Conversion Tool): Converts script.txt files to sceneData.js format
- **搜索功能修复程序** (Search Function Fix Tool): Fixes the scene search functionality by generating meta.js
- **用于GalTransl的文本转换程序** (Text Conversion Tool for GalTransl): Converts between script.txt and json formats for use with GalTransl project

Each folder contains a README_EN.txt file with detailed usage instructions.
