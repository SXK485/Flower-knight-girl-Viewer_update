Place ScriptToJson.py in the package.nw directory and run it via command line. It will generate a json_jp folder
containing converted json files ready for use with the GalTransl project. Simply place the entire json_jp folder in the custom project path under GalTransl, such as the default sampleProject.

Place ScriptToJson.py and the json_cn folder generated by the GalTransl project in the package.nw directory and run it via command line.
This will backfill the translated json from the GalTransl project to the corresponding script.txt files in the scenes folder.

GalTransl project repository: https://github.com/XD2333/GalTransl
