Place it in the package.nw directory and run it to generate a new meta.js. The data.json is updated once a month.
Only basic search functionality is fixed, i.e., searching scenes by English or Japanese names. Search conditions like CV, TAG, ARTIST, etc. are still unavailable,
because there's no reliable data source and maintenance is difficult.

PS: Only characters that exist in the local scenes directory will be fixed, not including undownloaded characters. It's recommended to update first, then fix the search.
