Using the automated translation project GalTransl with gpt-3.5-turbo and gpt-3.5-turbo-0613 models, completed over two weeks, up to 2023-11-16.
First run JsonToScript.exe, then run scriptConversion.exe, and replace the old sceneData.js with the generated one.
If needed, you can also run createMeta.exe to supplement the scene search index.
Due to varying translation quality from the model, if you're not satisfied with certain scene translations, you can modify individual scene json files and then proceed with the operations.
Tip: Before operating, please run update.exe to update scenes to the latest version. If needed, run ScriptToJson.exe to generate json files to backup the original text.
